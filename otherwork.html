Cloth Simulation - Fast Mass-Spring System (2018)

C++/OpenGL implementation based on Liu et al., Fast Simulation of Mass-Spring Systems (TOG 2013). Using implicit Euler, each time step solves for positions x (f = −∇E the elastic force, y the inertial prediction):

M(x−y) = h²f(x)  ⇔  minx ½(x−y)TM(x−y) + h²E(x)

The video compares three solvers:

  1. Local-global: Alternates parallel per-spring projection onto rest lengths with one sparse global solve, reusing a single Cholesky factorization since (M + h²L) is constant.
  2. Gradient descent: No Hessian, so iterations are cheap, but stiff springs force small steps and slow convergence.
  3. Newton's method: Uses the Hessian for fast convergence near the solution, at the cost of repeated factorization and instability when it is indefinite.
