Cloth Simulation - Fast Mass-Spring System (2018)

C++/OpenGL implementation of Liu et al., Fast Simulation of Mass-Spring Systems (TOG 2013). Using implicit Euler, each time step solves for positions x (f = −∇E the elastic force, y the inertial prediction):

M(x−y) = h²f(x)  ⇔  minx ½(x−y)TM(x−y) + h²E(x)

The video compares three solvers:

  1. Local-global: Alternates parallel projection of each spring onto its rest-length constraint with one sparse global solve. Since (M + h²L) is constant, its Cholesky factorization is computed once and reused.
  2. Gradient descent: Uses only the objective gradient. Each iteration is cheap, but stiff springs require small step sizes, slowing convergence.
  3. Newton's method: Uses the full Hessian (M + h²∇²E(x)). It converges quickly near the solution but requires repeated factorization and may diverge if the Hessian is indefinite.
